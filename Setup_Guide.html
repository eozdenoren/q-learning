Algorithmic Pricing — Setup Guide

Overview

You'll run a pricing simulation app on your own laptop. Two steps:

  1. Install Python (skip if you already have it)
  2. Download one file and run it — everything else is automatic
Already have Python? Open Terminal (Mac) or Command Prompt (Windows) and type python3 --version (Mac) or python --version (Windows). If you see Python 3.9 or higher, skip straight to Step 2.

1 Install Python

Go to python.org/downloads and download the latest version.

Mac

Open the downloaded .pkg file and follow the installer. That's it.

Windows

Run the downloaded .exe file. On the first screen, check the box at the bottom that says:

☑ Add python.exe to PATH

Then click Install Now.

2 Download and Run

  1. Click here to download run_app.py
    Google Drive will show a warning: "can't scan this file for viruses" — this is normal for any Python file. Click "Download anyway".
    Save it to your Desktop.
  2. Open Terminal (Mac) or Command Prompt (Windows).
  3. Paste the command below and press Enter:

Mac

python3 ~/Desktop/run_app.py

Windows

python %USERPROFILE%\Desktop\run_app.py

The script will automatically:

  1. Install the required Python packages (first time only, ~1 minute)
  2. Extract the app files
  3. Open the app in your browser
A browser window will open automatically showing the Q-Learning Demo. Use the sidebar on the left to navigate between pages, including Algorithmic Pricing.

Running Again Later

Just run the same command again. It will skip the installation step and launch immediately.

Mac

python3 ~/Desktop/run_app.py

Windows

python %USERPROFILE%\Desktop\run_app.py

Stopping the App

Go back to Terminal / Command Prompt and press Ctrl+C.

Troubleshooting

ProblemSolution
python3: command not found (Mac) Reinstall Python from python.org. Make sure to use the .pkg installer.
'python' is not recognized (Windows) You didn't check "Add to PATH" during installation. Uninstall Python, reinstall, and check the PATH box.
Browser doesn't open Open your browser and go to http://localhost:8501
Training takes a long time Normal — 20-60 seconds with default parameters. It runs on your laptop's CPU.
Permission denied (Mac) Try: chmod +x ~/Desktop/run_app.py then run again.
